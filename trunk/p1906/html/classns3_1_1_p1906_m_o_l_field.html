Detailed Description

Base class implementing the Field component of the P1906 framework, dedicated to the MOLECULAR Example.

This class implements persistence length as described in: Bush, S. F., & Goel, S. (2013). Persistence Length as a Metric for Modeling and Simulation of Nanoscale Communication Networks, 31(12), 815-824. http://dx.doi.org/10.1109/JSAC.2013.SUP2.12130014.

The Gnu Scientific Library (GSL) is required for these methods. All points and positions are in three dimensions comprised of a gsl_vector * of length three (x, y, z). All lines and segments are comprised of two points within a gsl_vector * of length size (x1, y1, z1), (x2, y2, z2). Lists of points or positions are in a gsl_matrix * of size n x 3. Each tube is comprised of a list of segments within a gsl_matrix * of size s x 6 -> s x ((x1, y1, z1), (x2, y2, z2)). A set of tubes is also a gsl_matrix * of size (s * t) x 6, where s is the number of segments and t the number of tubes. All random number are derived from gsl_rng *.

This vector gives the slopes of the three parametric equations describing the line between the points. AB = [ x, y, z] = [3 - 2, 7 - 4, -3 - 2] = [1, 3, -5]. Three parametric equations for a line are: x = x1 + at y = y1 + bt z = z1 + ct, So if we let A = (x1, y1, z1) x = 2 + t y = 4 + 3t z = 2 - 5t These three parametric equations describe the line connecting A and B. To see if C is on this line, we can substitute its x coordinate into the x parametric equation to find t: 1 = 2 + t, so t = -1.
